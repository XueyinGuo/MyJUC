# 2. 什么情况用什么锁
### 2.1 系统锁
执行时间长，线程数多
### 2.2 自旋锁
加锁代码执行时间短，而且线程数少下的情况

+  代码执行时间短：执行完的时间 < 内核态与用户态切换申请系统锁的时间，当然自旋效率更高
+ 如果虚拟机线程现在有几百个，虽然执行时间短，但是每个线程都要用 `while(true)`循环来耗费CPU，也是受不了这种资源消耗的
